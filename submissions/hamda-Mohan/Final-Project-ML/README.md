- Achieved **99.3% accuracy** with XGBoost as the best-performing model
- Developed REST API using **Flask** for real-time predictions
- Deployed and tested model endpoints (`/register`,`/login`,`/logout`,`/predict`)
- Included dataset, preprocessing scripts, trained models, API code and Frontend
